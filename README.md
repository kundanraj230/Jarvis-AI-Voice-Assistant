# 🤖 Jarvis AI Voice Assistant

An intelligent voice-controlled assistant built using Python that performs tasks based on user voice commands. Inspired by J.A.R.V.I.S., this project focuses on automation and seamless human-computer interaction.

---

## 🚀 Features

* 🎤 Voice Recognition
* 💬 Natural Language Processing
* 🌐 Open Websites & Search Google
* 🎵 Play Music
* 🗣️ Text-to-Speech Response
* ⚡ Task Automation
